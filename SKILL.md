# MallowMauve

## Overview

Use this skill to keep MallowMauve work calm, editorial, and brand-led. Treat each page like a private gallery experience, not a mass-market storefront.

## Workflow

1. Read [references/brand-brief.md](references/brand-brief.md) before making design or copy decisions.
2. Audit the current artifact and remove anything loud, crowded, sales-heavy, or visually generic.
3. Establish the hierarchy first: wordmark, hero, curated collection, craftsmanship note, and private enquiry.
4. Compose with generous whitespace, restrained typography, muted color, and a slow rhythm.
5. Finish with a polish pass for mobile balance, accessibility, and CTA softness.

## Interface Rules

- Prefer elegant serif headings and clean sans-serif body copy.
- Use a muted palette with ivory, mauve, beige, taupe, and charcoal.
- Favor large margins, thin dividers, and a measured editorial cadence.
- Use subtle motion only when it supports pace and atmosphere.
- Keep CTAs soft and discreet.

## Content Rules

- Write with restraint. Keep copy short, tactile, and emotionally quiet.
- Emphasize craftsmanship, provenance, materials, and small-batch curation.
- Prefer phrases like "Enquire Privately" or "Request Details" over hard-sell commerce language.
- If product imagery or content is missing, use tasteful placeholders and state the assumption in the handoff.

## Avoid

- Do not introduce bright colors, dense layouts, popups, countdowns, badges, or discount-driven UI.
- Do not default to generic ecommerce patterns such as star ratings, urgency copy, or oversized cart controls.
- Do not over-explain the brand. Let hierarchy, materials, and whitespace carry part of the story.
